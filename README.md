A minimalist, vibe-coded fitness tracker and virtual pet. Log your progress, earn credits, and keep your pixelated companion happy.

## Features
- **Pixel Pet:** A dynamic companion that reacts to your activity and care.
- **Weight Tracking:** Simple daily logging with progress visualization.
- **Workout Challenges:** Custom daily goals with swipe-to-delete functionality.
- **Hybrid Storage:** Automatic SQLite persistence when running locally, with LocalStorage fallback for web hosting.

## Quick Start
1. Start the backend server:
   ```bash
   python3 server.py
   ```
2. Open `weight-tracker.html` in your browser (defaults to `http://localhost:8000/weight-tracker.html`).

## Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript, Chart.js.
- **Backend:** Python (SimpleHTTP + SQLite).
